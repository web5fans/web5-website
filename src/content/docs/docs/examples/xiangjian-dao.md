The [Xiangjian DAO](https://xjdao.xyz) is a Web5-based social platform that leverages the AT Protocol for data storage and decentralized identities, providing a user-friendly interface for community engagement.

It is fully open-source:

1. [**atproto**](https://github.com/web5fans/atproto): AT Protocol implementation for the platform.

2. [**plc**](https://github.com/did-method-plc/did-method-plc): plc DID server for decentralized identities.

3. [**post_cache**](https://github.com/web5fans/post_cache): App view for caching posts and user data.

4. [**web5_deploy**](https://github.com/web5fans/web5_deploy): Deployment scripts for AT Protocol services.

5. [**social-app**](https://github.com/web5fans/social-app): Frontend for the Xiangjian DAO social application.

6. [**social-app-admin**](https://github.com/web5fans/social-app-admin): Admin frontend for managing the platform.

7. [**xiangjiandao-core**](https://github.com/web5fans/xiangjiandao-core): Backend core for Xiangjian DAO functionalities.
