[bbs.fans](https://bbs.fans) is a friendly community for Web5 enthusiasts to discuss, share, and learn about the latest developments in decentralized web technologies. It adapts AT protocol and Bluesky social app to build a community forum with `did:ckb` in a retro internet vibe.

It is fully open-source:

- [bbs](https://github.com/web5fans/bbs):	app view
- [bbs-fe](https://github.com/web5fans/bbs-fe):	app Front-end
- [bbs_deploy](https://github.com/web5fans/bbs_deploy):	deployment scripts
