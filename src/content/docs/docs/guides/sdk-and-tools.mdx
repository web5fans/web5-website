The web5 tech stack is still at an early stage. This page is an effort to document and share the SDKs and tools available for building decentralized applications with Web5.

## SDKs

- [@ckb-ccc/did-ckb](https://github.com/ckb-devrel/ccc/pull/337): A JavaScript SDK for creating, managing, and resolving CKB DIDs.
- [rsky](https://github.com/web5fans/rsky): A Rust SDK for building Web5 PDS, providing tools and libraries to interact with decentralized identities and data storage.

## Infra Tools

- [web5-indexer](https://github.com/web5fans/web5-indexer): An indexer for CKB DIDs that follows every DID transaction on the CKB blockchain and indexes every DID record through the path `<server url>\{:did}`.
- [Developer Console](https://console.web5.fans): A complete Web5 developer toolbox. Debug, explore, and build Web5 apps with full access to DID operations, PDS browser, and ATProto tools.
- [User Portal](https://me.web5.fans): Your entry point to Web5. Create your decentralized identity, discover apps, and manage your data—all without technical complexity.
- [Keystore](https://keystore.web5.fans): Securely manage your signing keys. Provides cryptographic signing capabilities for all Web5 applications via secure iframe isolation.

More tools and SDKs will be added here as the ecosystem grows. Stay tuned for updates!
